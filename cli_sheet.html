🖥️PuTTY
SSH Connection▾
  • putty.exe -ssh <user>@<host>SSH to host as user
  • putty.exe -ssh <host> -P <port>SSH on custom port
  • putty.exe -load "Session Name"Load saved session
  • putty.exe -ssh <host> -i key.ppkAuth with PPK key file
  • putty.exe -v -ssh <host>Verbose SSH debug
Telnet / Serial▾
  • putty.exe -telnet <host>Telnet connection
  • putty.exe -serial COM3 -sercfg 9600,8,n,1,NSerial @ 9600 baud
  • putty.exe -serial COM3 -sercfg 115200,8,n,1,NSerial @ 115200 baud
Port Forwarding▾
  • putty.exe -ssh <host> -L <lp>:<rh>:<rp>Local port forward
  • putty.exe -ssh <host> -R <rp>:localhost:<lp>Remote port forward
  • putty.exe -ssh <host> -D <port>SOCKS5 dynamic proxy
  • putty.exe -ssh <host> -NTunnel only, no shell
PLINK — CLI SSH▾
  • plink <user>@<host>Interactive SSH session
  • plink <user>@<host> "show ip route"Run single remote cmd
  • plink -batch -i key.ppk <host> <cmd>Non-interactive with key
  • plink -ssh -P 22 <host>Specify SSH port
PSCP / PSFTP▾
  • pscp <user>@<host>:/remote/file .Download file via SCP
  • pscp localfile <user>@<host>:/path/Upload file via SCP
  • pscp -r localdir <user>@<host>:/path/Recursive upload
  • psftp <user>@<host>Interactive SFTP session
Key Management▾
  • puttygen key.pem -o key.ppkConvert PEM to PPK
  • puttygen -t rsa -b 4096 -o new.ppkGenerate 4096-bit RSA key
  • puttygen key.ppk -O public-opensshExport OpenSSH public key
  • pageant key.ppkLoad key into SSH agent
Session & Logging▾

  • putty.exe -logfile session.log -ssh <host>Log session to file
  • putty.exe -sessionlog all.log <host>Full session logging
